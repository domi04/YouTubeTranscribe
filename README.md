# YoutTubeTranscribe

A simple tool to transcribe YouTube videos using OpenAI's Whisper model.

## Usage

Run the following command to transcribe a YouTube video:

```python
python youtube_transcribe.py "YOUTUBE_VIDEO_URL"
```

Replace `YOUTUBE_VIDEO_URL` with the actual URL of the YouTube video you want to transcribe.

The transcription will be saved in a text file named yt_audio.txt after the video title.
